   - Export `classifyBlock(text: string): OutputBlockType` as a standalone function (testable):
     - Error patterns: `/^(Error|error|ERROR|FATAL|panic|FAIL)[:!]/`, `/\b(traceback|exception|failed|denied)\b/i`
     - Tool call patterns: `/^(Read|Write|Edit|Bash|Glob|Grep|WebSearch|WebFetch)\(/`, `/^\s*\$ /`, `/^(Creating|Updating|Deleting|Reading) /`
     - Default: 'text'
   - Create `WebSocketStore` class using Svelte 5 `$state` runes:
     - State: `socket: WebSocket | null`, `connectionStatus: ConnectionStatus`, `activeSessionId: string | null`, `outputBlocks: OutputBlock[]`, `totalBytes: number`, `phaseStatuses: Map<string, PhaseStatus>`, `hasError: boolean`
     - Private: `cursor: number = 0`, `blockIdCounter: number = 0`, `retryCount: number = 0`, `retryTimer: ReturnType<typeof setTimeout> | null = null`, `decoder: TextDecoder | null = null`
     - `MAX_BLOCKS = 2000` constant
     - `connect(sessionId: string)`: disconnect first, create new TextDecoder with `stream: true`, build WS URL using `window.location` protocol/host, set connectionStatus to 'reconnecting', create WebSocket, wire onopen/onmessage/onclose/onerror handlers
     - `disconnect()`: clear retryTimer, close socket if open, set connectionStatus to 'disconnected', reset activeSessionId, set socket to null, set decoder to null
     - `send(message: ClientMessage)`: JSON.stringify and send if socket is open
     - `handleMessage(msg: ServerMessage)` (public for testing): switch on msg.type:
       - 'output': decode base64 to UTF-8 using TextDecoder with stream:true, split on newlines, classify each line, push OutputBlock(s), update cursor from offset + data length, cap blocks at MAX_BLOCKS
       - 'state': update phaseStatuses map for activeSessionId, derive PhaseStatus from session_state string ('running' -> 'working', 'exited' -> 'exited', 'spawning' -> 'working')
       - 'connected': set connectionStatus to 'connected', update totalBytes
       - 'error': push an error OutputBlock
       - 'metrics': store latest metrics (optional, for future use)
     - `scheduleReconnect(sessionId: string)`: exponential backoff (1s, 2s, 4s, max 30s), increment retryCount, set connectionStatus to 'reconnecting', setTimeout to call connect
     - `clearOutput()`: reset outputBlocks, cursor, blockIdCounter
   - Export singleton: `export const wsStore = new WebSocketStore()`
