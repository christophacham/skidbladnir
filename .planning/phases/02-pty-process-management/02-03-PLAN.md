  <action>
    **Step 1: Create metrics.rs.**

    `crates/agtxd/src/session/metrics.rs`:

    ```rust
    use serde::Serialize;
    use std::time::Instant;

    /// Snapshot of process resource usage at a point in time
    #[derive(Debug, Clone, Serialize)]
    pub struct MetricsSnapshot {
        pub cpu_percent: f32,
        pub rss_bytes: u64,
        pub uptime_secs: f64,
    }
    ```

    **Internal state for delta calculation:**
    ```rust
    /// Tracks previous CPU reading for delta calculation
    pub struct MetricsCollector {
        pid: i32,
        prev_cpu_ticks: u64,
        prev_wall_time: Instant,
        session_start: Instant,
    }
    ```

    **MetricsCollector::new(pid: u32, session_start: Instant) -> Self:**
    Store pid as i32, initialize prev_cpu_ticks to 0, prev_wall_time to Instant::now().

    **MetricsCollector::collect(&mut self) -> Option<MetricsSnapshot>:**
    1. Read /proc via `procfs::process::Process::new(self.pid)`. Return None if process doesn't exist.
    2. Get `stat = proc.stat().ok()?` and `statm = proc.statm().ok()?`.
    3. Calculate current CPU ticks: `stat.utime + stat.stime` (both u64).
    4. Calculate delta: `current_cpu - self.prev_cpu_ticks`.
    5. Calculate elapsed wall ticks: `self.prev_wall_time.elapsed().as_secs_f64() * procfs::ticks_per_second() as f64`.
    6. CPU%: `if elapsed_ticks > 0 { (delta as f32 / elapsed_ticks as f32) * 100.0 } else { 0.0 }`.
    7. Update prev_cpu_ticks and prev_wall_time.
    8. RSS: `statm.resident * procfs::page_size()` (page_size returns u64).
    9. Uptime: `self.session_start.elapsed().as_secs_f64()`.
    10. Return `Some(MetricsSnapshot { cpu_percent, rss_bytes, uptime_secs })`.

    Make metrics collection entirely fallible -- if /proc is inaccessible (containers), return None. Log a warning once via `tracing::warn!` with a flag to avoid spam.

    **Step 2: Add to session module.**

    In `crates/agtxd/src/session/mod.rs`, add `pub mod metrics;` and export `pub use metrics::{MetricsCollector, MetricsSnapshot};`.

    **Step 3: Write tests in `crates/agtxd/tests/metrics_tests.rs`.**

    **Step 2: Create background polling function.**

    In `crates/agtxd/src/session/metrics.rs`, add:

    ```rust
    pub async fn metrics_polling_task(
        pid: u32,
        session_start: Instant,
        metrics_cache: Arc<RwLock<Option<MetricsSnapshot>>>,
    ) {
        let mut collector = MetricsCollector::new(pid, session_start);

        // Initial delay to let process start
        tokio::time::sleep(Duration::from_secs(1)).await;

        loop {
            if let Some(snapshot) = collector.collect() {
                *metrics_cache.write().await = Some(snapshot);
            } else {
                // Process gone or /proc inaccessible
                break;
            }
            tokio::time::sleep(Duration::from_secs(5)).await;
        }
    }
    ```

    **Step 3: Integrate polling into SessionManager::spawn().**

    In `crates/agtxd/src/session/manager.rs`, after creating the SessionHandle in `spawn()`:
    1. Create `metrics_cache: Arc<RwLock<Option<MetricsSnapshot>>>` initialized to None.
    2. Spawn the polling task: `tokio::spawn(metrics_polling_task(pid, Instant::now(), metrics_cache.clone()))`.
    3. Store the JoinHandle and metrics_cache in the SessionHandle.

    **Step 4: Add get_metrics method to SessionManager.**

    ```rust
        let sessions = self.sessions.read().await;
        let handle = sessions.get(&id)?;
        handle.metrics.read().await.clone()
    }
    ```
